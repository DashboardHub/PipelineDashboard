## Setup

1. Install dependencies

```
php composer.phar install
```

2. Update parameters

```
/app/config/parameters_dev.yml
```

Database parameters are important: host, username, password etc...

## Run app

1. `make symfony.server`

2. Then go to `http://localhost:8000`
